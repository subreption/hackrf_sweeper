Detailed Description

Library initialization & exit

The sweeping API needs an initialized state with frequency ranges and output mode. A hackrf_sweep_state must be previously allocated and passed to the hackrf_sweep_init or optionally the hackrf_sweep_easy_init function. The output mode must be chosen first via hackrf_sweep_set_output and the frequency ranges must be configured with hackrf_sweep_set_range afterwards.

Sweep control

The sweep can be started via hackrf_sweep_start and stopped with hackrf_sweep_stop. A stopped sweep can run again with no additional configuration. hackrf_sweep_close must be used when the sweep state is no longer needed. The state will be released and invalidated.
